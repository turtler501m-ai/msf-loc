## 프로젝트 요약

**MSF(스마트서식지)** — 대리점용 모바일 고객 서비스 신청 시스템.
M포탈(MCP) 동일 기능을 신규 개발. M플랫폼/KT(KOS) 연동 스펙 동일 구현.

```
msf-web (Vue 3, port 9480) →[/api proxy]→ msf-api (Spring Boot, port 8081)
    → M전산 DB (PostgreSQL/Oracle)
    → M플랫폼 HTTP → KT(KOS)
```

---

## 도메인 패키지 (msf-api)

루트 패키지: `com.ktmmobile.msf`

| 패키지 | 역할 | API prefix |
|--------|------|------------|
| `formComm` | 공통 (가입자조회, 동시처리) | `/api/v1/{domain}/join-info` |
| `formSvcChg` | 서비스변경 (부가서비스·요금제) | `/api/v1/service-change/*`, `/api/v1/addition/*` |
| `formOwnChg` | 명의변경 | `/api/v1/ident/*` |
| `formSvcCncl` | 서비스해지 | `/api/v1/cancel/*` |
| `common.mplatform` | M플랫폼 HTTP 어댑터 | - |
