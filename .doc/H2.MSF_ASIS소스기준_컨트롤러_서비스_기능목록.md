| 기호 | 의미 |
|------|------|
| ✅ | TOBE 구현 완료 (`@RestController` + `/api/v1/*`) |

### TOBE 불필요 판단 기준
- `public String` 반환 (JSP 화면 경로 반환) → Vue Router가 화면 전환 담당
- `HttpSession session` 의존 세션 기반 처리 → TOBE에서 JWT/토큰 방식으로 변경
- 팝업·배치·탭별 목록 (`Pop.do`, `Tab*.do`, `batch/`) → TOBE 범위 외
- KT 결합 계열 (`combineKt`, `childCertify` 등) → 아무나SOLO만 TOBE 범위
