## 2. 전체 플로우

**20260310**: 가입유형 선택(S104050101)이 **고객(S104010101) 화면 하단**으로 통합됨.

```
고객(S104010101 + 가입유형 하단) → 상품(S104020101) → 동의(S104030101) → 신청완료(S104040101)
```

- **처리 흐름(설계서)**  
  신청서 작성 완료 → 신청서 등록 → 리포트 이미징 처리 → 이미징 시스템 전송 → M플랫폼 서비스해지 → KOS 서비스해지 → K-NOTE(신청서). 서비스해지 요청·신청서 전송. M전산으로 데이터 보낼지 검토(IT부서).  
- **비고**  
  서비스해지에 따른 잔여 요금 조회 API(m플랫폼) 확인(IT부서) → 제공 가능 시 조회로 처리.
