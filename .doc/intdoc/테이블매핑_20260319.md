# MSF ↔ MCP 테이블 매핑 정의서

> 작성일: 2026-03-19
> 목적: ASIS(MCP/NMCP) → TOBE(MSF) 테이블명 변환 기준 정의

---

## 변환 규칙

| ASIS 출처 | 접두사 | TOBE 처리 |
|----------|--------|----------|
| M포탈 DB (MCP 포털) | `MCP_*`, `NMCP_*` | `MSF_*` 로 접두사 교체 |
| M플랫폼 DB (MSP) | `MSP_*` | `MSP_*@DL_MSP` DB링크 그대로 사용 |
