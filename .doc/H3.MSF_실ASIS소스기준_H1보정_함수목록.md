# H3. MSF 실ASIS소스기준 H1 보정 함수목록

> **ASIS 소스 기준:** `mcp/mcp-portal-was/`
> **TOBE 참조:** `msf-be-form-api/domains/form/`
> **작성일:** 2026-04-13
> **목적:** `H1` 문서를 실제 ASIS 소스와 대조한 결과를 정리하고, E1 기준 문서에서 생략되거나 오해될 수 있는 실제 public 메서드 기준 목록을 보조 문서로 제공

---

## 요약

- `H1`은 `E1` 기반 개발대상 매핑표라서 실제 ASIS public 메서드 전체 목록과 1:1 대응하지 않음
- 실제 소스 기준으로 보면 `PauseController`, `CustRequestController`, `MyShareDataController` 등은 `H1`보다 범위가 넓음
- `MyNameChgController.userInfoChk`는 ASIS 1:1 함수가 아니라 TOBE 보완 대응 항목으로 보는 것이 타당함

## 3. 문서 사용 가이드

- `H1`: E1 기준 개발대상 매핑표
- `H2`: 실제 ASIS 소스 기준 TOBE 필요/불필요 분류표
- `H3`: `H1`을 실제 ASIS 소스와 대조할 때 참고하는 보정 메모와 실제 함수 기준 보조 목록

실제 구현 누락을 확인할 때는 `H2`, E1과의 요구사항 대응을 볼 때는 `H1`, 두 문서 간 해석 차이를 설명할 때는 `H3`를 우선 사용한다.
