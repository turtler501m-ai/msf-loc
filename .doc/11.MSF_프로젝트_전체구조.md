## 1. 워크스페이스 전체 구성

```
C:/MSF-LOC/workspace/
├── CLAUDE.md                   # 프로젝트 가이드 (AI 개발 지침)
├── .claudeignore               # Claude 컨텍스트 제외 설정
├── .claude/                    # Claude Code 설정 (skills, commands)
├── .doc/                       # 개발 문서 (마크다운)
├── mcp/                        # 기존 M포탈 (레거시, ASIS 참조용)
├── msp/                        # M플랫폼 연계처리 (레거시, 참조용)
└── msf/                        # 신규 스마트서식지 (TOBE, 개발 대상)
```

### 역할 구분

| 폴더 | 역할 | 사용 목적 |
|------|------|----------|
| `mcp/` | 기존 M포탈 전체 소스 | ASIS 참조 전용 (수정 금지) |
| `msp/` | M플랫폼 연계처리 레거시 | 참조용 (연동규격은 .doc에 정리됨) |
| `msf/` | 신규 스마트서식지 | 개발 대상 (msf-api + msf-web) |
| `.doc/` | 개발 분석·설계 문서 | 기능 명세, 연동 스펙 |
| `.claude/` | Claude Code 설정 | 스킬 가이드, 슬래시 커맨드 |

## 2. 연동 아키텍처

```
[대리점 앱]
    │
    ▼
msf-web (Vue 3, port 9480)
    │  /api → Vite proxy
    ▼
msf-api (Spring Boot, port 8081)
    │
    ├── M전산 DB (PostgreSQL 로컬 / Oracle 운영)
    │
    └── M플랫폼 HTTP (juice.url → mPlatform/serviceCall.do)
            └── KT(KOS) 시스템
```

cd msf/msf-api

### Windows API 테스트

```js
// curl 대신 Node.js 사용 (한글 UTF-8 인코딩 문제 회피)
const http = require('http');
const body = JSON.stringify({ name: "홍길동", ctn: "01012345678" });
const req = http.request({
  hostname: 'localhost', port: 8081,
  path: '/api/v1/addition/current', method: 'POST',
  headers: { 'Content-Type': 'application/json; charset=utf-8',
             'Content-Length': Buffer.byteLength(body) }
}, res => { let d = ''; res.on('data', c => d += c); res.on('end', () => console.log(d)); });
req.write(body); req.end();
```
