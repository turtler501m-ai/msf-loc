## 2. 서비스해지 신청서 작성 흐름 개요

### 고객 단계
- **고객유형** : 내국인 / 미성년자(19세 미만) / 외국인(Foreigner) / 외국인(Foreigner) 미성년자 / 법인 / 공공기관
- 법인/공공기관만 방문고객(본인/대리인) 선택 가능
- 성인 : '본인' 고정, 미성년자 : '대리인' 고정
- **가입자 정보** : 이름, 생년월일 / 법인등록번호
- **서비스해지 휴대폰번호** 체크 — 고객유형 '미성년자'인 경우만 해당
- **법정대리인 정보** (미성년자 해당) : 이름, 신청인과 관계, 주민등록번호, 휴대폰번호, 안내 및 동의사항
- **대리인 위임 정보** (방문고객이 '대리인'인 경우) : 이름, 생년월일, 신청인과 관계, 연락처
- **연락처 정보** : 연락처, 이메일주소, 주소
- **약관 동의** : 필수/선택 — 고객유형에 따라 해당 약관 표시
- **해지 후 사용 여부** : 통신사 중 선택

### 상품 단계
- **해지 정산** : 사용요금, 위약금, 단말기 할부금
- **메모** 입력

### 동의 단계
- **고객 안내사항** 확인
- **신청서 확인** : 가입정보 확인, 서명(가입자/대리인 서명) — 팝업으로 최종 확인
- **신청서 처리** : 신청서 등록 → 리포트 이미징 처리 → 이미징 시스템 전송 → 서비스해지 M전산 전송 → 신청서 K-NOTE 전송

### 완료 단계
- **신청서 교부** : 열람, 발송

## 3. 서비스해지 신청서 처리 흐름

```
스마트서식지
  └─ 신청서 작성 완료
       └─ 신청서 등록
            ├─ 이미징 시스템 → 리포트 이미징 처리 (리포트 서식)
            ├─ M플랫폼 → 리포트 이미징 전송 → M전산
            ├─ 서비스해지 → 서비스해지 요청 → K-NOTE
            └─ 신청서 → 신청서 전송 (첨부서류 포함)
```
