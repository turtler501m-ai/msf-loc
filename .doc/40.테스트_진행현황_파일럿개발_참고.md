## 테스트 환경 설정

### 서버 실행

```bash
# 빌드
cd msf/msf-api
"/c/Program Files/JetBrains/IntelliJ IDEA 2025.3.3/plugins/maven/lib/maven3/bin/mvn" \
  -DskipTests package

# 실행 (백그라운드, 로그 파일 출력)
"/c/Program Files/Java/jdk-21.0.10/bin/java" -Dfile.encoding=UTF-8 \
  -jar target/msf-api-0.1.0-SNAPSHOT.jar > /c/MSF-LOC/workspace/msf-server.log 2>&1 &

# 로그 확인
tail -f /c/MSF-LOC/workspace/msf-server.log
```

### application.properties 로컬 설정

```properties
SERVER_NAME=LOCAL                      # M플랫폼 Mock 응답
msf.join-info.mock-when-no-db=true     # DB 없을 때 Mock 인증 허용 (로컬 전용)
```

> ⚠️ `msf.join-info.mock-when-no-db=true` 는 로컬 개발 전용. 운영 배포 시 반드시 주석 처리.

### Mock 테스트 계정

| 항목 | 값 |
|------|-----|
| 이름 | 홍길동 |
| 휴대폰 | 01012345678 |
| ncn (Mock 반환) | 100000001 |
| custId (Mock 반환) | MOCK001 |

### 통합 테스트 스크립트 실행

```bash
# 서버 기동 후 실행
node .doc/reference/script/test_api.js
```

전 도메인 13개 케이스를 자동으로 검증. `✅ PASS` / `❌ FAIL` 로 결과 출력.
